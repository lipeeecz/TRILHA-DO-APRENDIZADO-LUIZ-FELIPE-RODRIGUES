Hiperlinks

Tambem chamados de LINKS. Que são uma referência dentro de diversos documentos em hipertexto para outras partes desse documento.
Os links podem ser:

  • Remotos
  • Internos
  • Locais
  • Com endereço de correiou ou arquivo

Fazer a marcaçao de um Hiperlinsks é utilizada por uma tag a e uma propriedade href que faz referencia com o conteudo desejado.

Tipos de Hiperlinks

  • Hiperlinks externos ou remotos:são os que fazem referencia com sites ou paginas da web.
  • Hiperlinks internos:Os links internos sao aqueles que apontam a um lugar dentro da mesma pagina, possibilitando acessar diferentes paragrafos e seçõe
  • Hiperlinks locais: É os que apontamm para outras paginas dentro de um mesmo site
  • Hiperlinks em imagens:Se realiza usando uma tag de imagem com a tag de hiperlinks
  • Hiperlinks para dowloads de arquivos:São links que apontam para arquivos e possibilitam realizar o seu dowload deste arquivo
  • Hiperlinks para endereço eletronico: Eles apontam para um endereço eletronico
