Branches e Hospedagem

Os Branches são ramificaçoes de um projeto de software ou documento, tendo com objetivo o desenvolvimento paralelo de:

  • novas de funcionamentos
  • Coreção de problemas
  • alteração de codigo

Podemos colocar o projeto na Branch Master ou Main que é o braço principal do projeto de repositotio e no Branch Secundaria que é o braço secundario de um projeto em um repositorio. As equipes podem trabalhar paralelamente na branch master, sem fazer alteraçoes, a mesclagem é feita depois de você ter certeza que está tudo certo, assim a sua alteraçao vai ficar salva na branch master.
