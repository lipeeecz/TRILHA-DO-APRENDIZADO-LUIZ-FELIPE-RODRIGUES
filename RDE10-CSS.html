Cascading Style Sheets

São folhas de estilo em cascatas, com o objetivo de realizqar a estilização do conteudo de documentos Html, podemos definir: cores, fontes, bordas, espacamento, posicionamneto, entre outros.

Componetes

Seletores que selecionam elementos html com base no nome do elemento ao qual o etilo sera aplicado.
Propriedades é o aspecto especifico de elementos que você quer usar como estilo.
Valoress sendo a configuração aplicada ao attributo.

Existem 3 maneiras de utilizar o CSS.

Inline
Os codigos são definidos dentro de uma tag Html e afetam somente a tag que foi definida.
Incorporado ou Internal
Os codigos CSS são definidos no cabeçalho, dentro da tag HEAD e afetam somente o documento em que foi definido.
Externo ou External
Os codigos CSS são definidos em um arquivo externo e afetam todos os documentos que estão linlinkados ao CSS.
