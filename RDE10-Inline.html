Cascading Style Sheets

Definição

Cascading Style Sheets - Folhas de Estilo em Cascata- tem por objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamento, alinhamentos, bordas dentre outros.

Componentes

  • Seletores- seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado
  • Propriedades- o aspecto específico que você quer usar como estilo Valores - a configuração aplicada ao atríbuto.

Formas de utilização

Inline

Os códigos CSS são definidos dentrod e uma tag HTML. Afetam somente a tag em que foi definido.

Incorporado ou Internal

Os códigos CSS são definidos no cabeçalho, dentro da tag head. Afetam somente o documento em que foi definido.

Externo ou external

Os códigos CSS são definidos em um arquivo externo. Afetam todos os documnetos que estão linkados ao CSS.
