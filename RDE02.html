Imagens.
É a representação visual, de uma pessoa,objeto. A imagem digital é uma imagem bidimisioal usando numeros binarios codificados.

Formatos de imagem.
Sao padroes de codificaçoes, compressao e definiçoes, que determinam o tipo do arquivo, são eles: .BMP, .TIFF, .GIF, .JPEG, .PNG dentre outros modelos.
As imagens sao importantes, devemos tomar cuidado com o tamanho delas, pois da diferença na hora de carregar seu site, deixando mais lento ou nao.

Licenças e Repositório de Imagens.
Para nao quebrar as regras e ser processado devemos usar as licenças Creative Commons, são licenças publicas que nos permite a distribuiçao gratuita para uma obra protegida por direitos autorais.Passo a passo de como conseguir a licença:
Podendo utilizar uma busca de imagens no google, escreve a palavara imagem e aplica os filtros, entrando em ferramentas, escolhendo o tipo de licença, marca a que voce escolheu(Creative Commons)assim ele exibe somente as imagens que tem esse tipo de licença.Outra forma é usando os repositórios gratuitos, entre eles temos: Unsplash, Wikimedia Commons, Pixabay, Rawpixel, FreePixel, Vintage Stock Photos.OBS: Leia sempre os termos de cada um, para garantir que voce pode usar.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

Marcação de Imagens

As tags de imagens e seus parametros definem as imagens e suas propriedades dentro do documento Html.
Tags figure e figcaption, foram definidas a partir do HTML 5, ajuda para a melhor organizaçao do documento.

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee.

A HTML é utilizada para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo , e que podem ser vinculados através de hyperlinks.

Marcação de Favicon

Ela é usada para a definição de ícones de favoritos que aparece na barra de titulos, a marcação é colocada nos itens de cabeçalho.
A sua sintaxe é:1o link: REL "icon" que mostra que é um icone, HREF sendo a referência para o icone(onde ele esta salvo),TYPE que é o tipo da imagem.
Os sites para encontrar os icones são: Icon aqrchive, Favicon.cc e Favicon.io
